# Bank Loan Analysis Dashboard
## Bank Loan Analysis Dashboard
### Overview

#### This project analyzes bank loan data to provide insights into lending performance, borrower behavior, and risk assessment. The dashboard enables data-driven decisions by tracking key financial metrics and loan trends.

### Objective

#### To build an interactive dashboard that helps monitor:

#### • Loan applications and growth trends
#### • Funded vs received amounts
#### • Borrower risk (DTI, interest rate)
#### • Good vs Bad loan performance
### Key Metrics
#### • Total Applications: 38.6K
#### • Funded Amount: $435.8M
#### • Amount Received: $473.1M
#### • Avg Interest Rate: 12.05%
#### • Avg DTI: 13.33%
### Insights
#### ✅ 86.18% loans are performing (Good Loans)
#### ⚠️ 13.82% loans are at risk (Bad Loans)
#### 📈 Loan applications show steady monthly growth
#### 💳 Major loan purposes include credit card & debt consolidation

### Business Value

#### This dashboard helps:
#### • Banks to reduce risk exposure
#### • Identify high-risk borrowers
#### • Improve loan approval strategies
#### • Optimize financial performance
