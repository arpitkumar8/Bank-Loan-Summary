###Bank Loan Analysis Dashboard
📊 Project Overview

This project focuses on analyzing bank loan data to generate actionable insights for improving lending decisions and monitoring portfolio performance. The dashboard provides a comprehensive view of loan applications, funding, repayments, and risk segmentation (Good vs Bad Loans).

The goal is to enable data-driven decision-making in banking and financial services.

🎯 Problem Statement

The objective is to build a Bank Loan Report Dashboard that helps:

Track loan application trends over time
Monitor funding and repayment performance
Analyze borrower financial health (DTI, Interest Rate)
Identify Good Loans vs Bad Loans
Support strategic lending decisions
